# Admin Admin Panel

Admin panel for the ai-web-learning monorepo.

## Setup

1. Copy `.env.example` to `.env` and fill in your Supabase credentials
2. Run `npm install`
3. Run `npm run dev` for development
4. Run `npm run build` for production build

## Deployment

Deploy to Vercel with `vercel deploy --prod --yes` from the admin directory.

## Tech Stack

- Vite + React 19 + TypeScript
- Tailwind CSS 4
- React Router 7
- Supabase
- Zustand for state
