TIRBEO PLATFORM — COMPLETED WORK
===================================
Last updated: 2026-06-30

ARCHITECTURE
------------
Monorepo (npm workspaces) with 8 apps + shared packages:
- chat/         — Real-time messaging client
- about/        — About page (company story, team)
- admin/        — Admin panel (black/white UI, Supabase-driven)
- dashboard/    — User dashboard
- accounts/     — Account settings
- blog/         — Blog
- chatlanding/  — Chat landing/marketing page
- docs/         — Documentation (Supabase-driven content)

All apps: Vite + TypeScript + Tailwind CSS 4 + React 19 + Supabase.
Single supabase project for all apps.

ADMIN APP (admin/)
------------------
5 core pages, all black/white theme:
- Dashboard      — Stats overview (articles, categories, admins)
- Site Config    — Manage site name, SEO, social links, logo
- Docs           — CRUD for doc_categories + doc_articles
- Content Approvals — Approve/reject pending content
- Admins         — Manage admin_users (roles: super_admin/admin/manager/editor/viewer)
- Users          — List all registered auth.users, add to admin_users

Key files:
- src/App.tsx           — Router with role-protected routes
- src/AdminLayout.tsx   — Sidebar layout, role-based nav filtering
- src/pages/*.tsx       — 6 page components
- src/lib/session.ts    — Auth session management (localStorage)
- src/lib/useAuth.ts    — React hook for auth state
- src/lib/roles.ts      — Role hierarchy + permission helpers
- src/lib/content.ts    — CRUD helpers + typed interfaces
- src/lib/supabase.ts   — Supabase client singleton

DOCS APP (docs/)
----------------
Supabase-driven documentation with local fallback:
- Sidebar built from doc_categories + doc_articles tables
- PageRenderer fetches from Supabase, falls back to content-local.ts
- Hash-based routing (/#article-slug)
- Clean black/white layout, markdown renderer with table support

Key files:
- src/App.tsx                          — Fetches categories/articles on mount
- src/components/PageRenderer.tsx      — Supabase primary + local fallback
- src/components/layout/Sidebar.tsx    — Category nav with search
- src/lib/content.ts                   — Supabase queries
- src/lib/content-local.ts             — 11 hardcoded doc pages fallback
- src/lib/markdown.tsx                 — Markdown renderer
